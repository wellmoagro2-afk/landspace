# Mapas dos Briefings

Este diretório contém os arquivos de mapas para download dos briefings.

## Arquivos esperados:

- `amazonia-soberania-map.png` (ou .pdf/.geojson/.zip) - Mapa do briefing sobre Amazônia
- `petroleo-oriente-medio-map.png` (ou .pdf/.geojson/.zip) - Mapa do briefing sobre Petróleo

## Formatos suportados:

- PNG/JPG para visualização
- PDF para documentos
- GeoJSON para dados geoespaciais
- ZIP para arquivos comprimidos

## Nota:

Para produção, substitua os arquivos placeholder por mapas reais gerados a partir de análises geoespaciais ou criados em ferramentas como QGIS, ArcGIS, ou plataformas online.

## Estrutura de URLs:

- `/briefings/maps/amazonia-soberania-map.png`
- `/briefings/maps/petroleo-oriente-medio-map.png`
